Hey — welcome. This is my corner of the internet.

## Why This Site Exists

I've had a few portfolios over the years. Static HTML pages, Wix experiments, half-finished React apps. This one is different — it's built to last.

I wanted a place where I could:

- **Show my projects** in more detail than a GitHub README allows
- **Write about what I'm building** — the failures, the breakthroughs, the weird edge cases
- **Keep everything in one place** — no more scattered links and forgotten repos

## What You'll Find Here

- **Projects** — full-stack apps, mobile builds, Unity games, AI tools
- **Blog** — dev logs, lessons learned, and whatever else I feel like writing
- **About** — the story, the skills, the experience
- **Contact** — if you want to work together or just say hi

## A Quick Note on the Design

I wanted this site to feel like a terminal crossed with a design portfolio. Dark mode by default (but light mode if you prefer), scroll animations that don't get in the way, and a focus on content over chrome.

Everything is built with Next.js and Tailwind CSS, deployed on Netlify. The code is [open source on GitHub](https://github.com/Sasen12/personal-site).

Thanks for stopping by.
