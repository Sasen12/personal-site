Android 17 started rolling out to Pixels today, and for once, the update isn't just a security patch with a new number. This is a genuinely feature-rich release that changes how I use my phone.

## What's New

- **System-wide Bubbles** — floating multitasking windows for any app
- **Screen Reactions** — your phone detects on-screen content and suggests contextual actions
- **Revamped notification stack** — grouped intelligently by priority, not chronologically
- **Battery Share 2.0** — reverse wireless charging that actually works at usable speeds

## Screen Reactions Are Smarter Than They Sound

The demo showed it recognizing a concert ticket screenshot and offering to add it to calendar *before you even open Photos*. It's not just OCR — it understands intent. This kind of proactive assistance is where Android should've been years ago.

## The Real Story

Android 17 isn't about any single feature. It's about Google finally treating the OS as a *platform for intelligence* rather than just an app launcher. The under-the-hood changes to the Machine Learning core are more important than any UI tweak.

Devs should pay attention to the new ML Kit 3.0 APIs — they let you piggyback on the same on-device models powering Screen Reactions. I've already started porting a side project to use them, and the inference speed is impressive.

First impressions: this is Android's best release since 12. Let's see how the non-Pixel OEMs handle the update.
